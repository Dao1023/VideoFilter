# VideoFilter 视频过滤器

## 介绍

忍无可忍，我要过滤掉B站的垃圾视频

B站垃圾视频的特点是时间，我们取小于 10 分钟和大于 30 分钟的给过滤掉

## 使用方法

安装油猴脚本插件 TamperMonkey

在油猴脚本中「添加新脚本」

复制 main.js 代码到油猴脚本

本代码随便拿去用
